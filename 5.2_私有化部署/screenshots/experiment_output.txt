============================================================
  5.2 Model Deployment Guide
  部署方案 (2026-06-29)
============================================================

[方案一] Ollama 本地部署
  1. 安装 Ollama: https://ollama.com/download
  2. 拉取基座模型:
     ollama pull qwen3:1.7b
  3. 创建 Modelfile (载入LoRA权重):
     FROM ./models/Qwen3.7-1.7B-Instruct
     ADAPTER ./models/lora/adapter
  4. 构建自定义模型:
     ollama create kb-agent-v1 -f Modelfile
  5. 启动服务:
     ollama serve
  6. 测试:
     curl http://localhost:11434/api/generate -d '{"model":"kb-agent-v1","prompt":"What is MCP?"}'
