## 通用技术规范

| 项目 | 规范 |
|------|------|
| **后端** | `matplotlib.use('Agg')` |
| **DPI** | 200 |
| **英文字体** | Times New Roman（通过 `plt.rcParams['font.serif']` 设为首选） |
| **中文字体** | `FontProperties(fname=r'C:\Windows\Fonts\msyh.ttc', size=N)` — 所有中文文本必须使用此方式，**禁止**用 `fontsize=` 写中文 |
| **箭头样式** | `FancyArrowPatch(arrowstyle='->,head_width=3,head_length=4', mutation_scale=8, lw=1.5)` |
| **盒子样式** | `FancyBboxPatch(boxstyle='round,pad=0.06', edgecolor=border_color, linewidth=1.8, alpha=0.9)` |
| **保存参数** | `plt.savefig(path, dpi=200, bbox_inches='tight', facecolor='white', pad_inches=0.3)` |
| **颜色方案** | 海军蓝标题 `#1a478a`，暖色(橙/黄)表示输入/思考，冷色(蓝/紫)表示处理/输出，绿色表示成功/完成，红色表示警告/错误 |
| **中文标签** | 最小字号 11pt（标题 14-15pt 加粗，子标题 13pt 加粗，正文 11pt，辅助标注 8-9pt） |
| **英文/公式** | 使用 mathtext 或 TNR 渲染 |

### 辅助函数参考

```python
import matplotlib; matplotlib.use('Agg')
import matplotlib.pyplot as plt
from matplotlib.patches import FancyBboxPatch, FancyArrowPatch
from matplotlib.font_manager import FontProperties
import numpy as np

plt.rcParams['font.family'] = 'serif'
plt.rcParams['font.serif'] = ['Times New Roman']
MSYH = r'C:\Windows\Fonts\msyh.ttc'
AK = dict(arrowstyle='->,head_width=3,head_length=4', mutation_scale=8, lw=1.5)

def cj(size): return FontProperties(fname=MSYH, size=size)

def bx(ax, cx, cy, w, h, color, border='#1a478a'):
    return ax.add_patch(FancyBboxPatch(
        (cx - w/2, cy - h/2), w, h, boxstyle='round,pad=0.06',
        facecolor=color, edgecolor=border, linewidth=1.8, alpha=0.9))

def ar(ax, x1, y1, x2, y2, dashed=False):
    a = dict(AK)
    if dashed: a['linestyle'] = 'dashed'
    ax.add_patch(FancyArrowPatch((x1, y1), (x2, y2), **a))

def save(path):
    plt.savefig(path, dpi=200, bbox_inches='tight', facecolor='white', pad_inches=0.3)
    plt.close()
```
