# 课程配图 (Course Illustrations)

本目录存放课程相关的配图文件，包括：

- 架构图（Architecture Diagrams）
- 流程图（Flow Charts）
- 示意图（Conceptual Diagrams）
- 对比图（Comparison Charts）

## 配图要求

- 格式：PNG 或 SVG（矢量图优先）
- 分辨率：不低于 1920x1080
- 命名规范：`{章节号}_{内容描述}.png`，如 `3.1_transformer_architecture.png`
- 所有配图应包含中文标注

## 配图生成建议

可使用以下工具生成课程配图：
- Draw.io / Excalidraw（架构图、流程图）
- Matplotlib / Plotly（数据图表）
- Mermaid（代码生成图表）
- AI 绘图工具（DALL-E、Midjourney）用于概念示意图
