# 大模型应用开发课程 (LLM Application Development)

## 课程概述

本课程面向具备 Python 基础及深度学习入门知识的学习者，系统讲解大语言模型（LLM）应用开发全流程。课程涵盖 Prompt Engineering、API 调用、RAG 系统、Agent 开发、模型微调、私有化部署及工程化实践，聚焦国产大模型（Qwen3、DeepSeek V4 等）与开源框架（LangChain、LangGraph 等）的实战应用。

## 全局配置

所有实验通过 `config.py` 快捷入口使用统一的模型配置:

```python
from config import get_client, verify_config

verify_config()                     # 诊断API配置
client = get_client()               # 自动选择可用提供商
client = get_client("deepseek")     # 或指定
result = client.chat("你好")       # 统一接口
```

配置模块位于 [1.2.2 实验模型配置](1.2.2_实验模型配置/), 详见该章 README。

## 课程文档结构

每章 README.md 统一包含以下章节：

1. **课程目标** — 3~5 条可衡量的学习目标
2. **背景介绍** — 技术产生背景、发展历程、应用场景（约 500 字）
3. **基础概念** — 核心理论，含公式、流程图、架构图
4. **环境准备** — Python 版本、依赖库、API Key、GPU 要求
5. **实践项目** — 与课程主题直接相关的实践项目设计
6. **实验步骤** — Step-by-step 操作说明、完整代码、代码解释
7. **实验结果** — 真实运行的控制台输出、图表、截图
8. **结果分析** — 结果解读、预期对比、问题分析、优化方向（不少于 300 字）
9. **扩展学习** — 参数调优、性能优化、部署方案、横向对比、推荐阅读（300~500 字）

## 技术栈

### 国产大模型
- **DeepSeek V4 Pro** — 1.6T MoE，37B活跃参数，推理+代码最强（¥0.87/1M输出）
- **DeepSeek V4 Flash** — 284B/13B活跃，极致性价比（¥0.28/1M tokens）
- **DeepSeek-R1-0528** — 最新推理增强模型，CoT深度思考
- **Qwen3.7-Max** — 阿里云旗舰，1M上下文，多模态+Agent，中文最强
- **Qwen3.7-MoE** — 48B活跃/200B总参数，单卡可部署
- **GLM-5.2** — 智谱AI最新（2026.6.17），Agent能力T0级
- **Kimi K2.7** — 月之暗面，256K无损上下文，92%召回率
- **豆包 Seed 2.0 Pro** — 字节跳动，多模态评测90.66分登顶

### 开源框架
- **LangChain / LangGraph** — LLM 应用与 Agent 编排
- **MCP (Model Context Protocol)** — AI Agent 标准化协议，Linux Foundation 托管，10,000+ 服务器
- **FAISS** — Facebook 开源向量检索库
- **BGE-M3 / bge-small-zh-v1.5** — BAAI 中文 Embedding 模型
- **vLLM / Ollama** — 模型推理与私有化部署
- **LLaMA-Factory / SWIFT** — 高效微调框架（LoRA/QLoRA）
- **Dify / FastGPT** — 开源 RAG/Agent 低代码平台

### 开发环境
- **Python** 3.10+
- **Jupyter Notebook / JupyterLab**
- **OpenAI SDK** (兼容接口调用国产模型)
